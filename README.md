### Software

| Projeto | Descrição | Stack Principal |
|---------|-----------|-----------------|
| **Landing Page — Visita Técnica Equinix** | Inscrição, controle de vagas em tempo real e pagamento via Mercado Pago para visita técnica da Braz Cubas ao Data Center Equinix. E-mail com certificado pós-pagamento. | React 19 · Node.js · MySQL · Mercado Pago · Socket.io · Redis |
| **Rota — Planejamento de Viagens com IA** | Plataforma completa de viagens assistida por IA. Sugestões personalizadas, timeline, calendário, mapa interativo, compartilhamento de roteiro, chat contextual e check-in de atividades. | Next.js 16 · OpenAI · Google Maps · Framer Motion · dnd-kit |
| **Where — Rastreamento GPS via Bluetooth** | Coleta e visualização de posicionamento em tempo real via Bluetooth LE. Mapa com Street View, histórico, status online/offline e acesso via Google OAuth. | Next.js 16 · Bluetooth LE · Google Maps API · Street View API |
| **Dronec — Marketplace para Pilotos de Drone** | Plataforma institucional e sistema de gestão para o mercado de drones. Conecta pilotos a clientes, portfólio de mídia e feed social interativo. | React 18 · Node.js · MySQL · Socket.io · Redis · Sharp |
| **Eye — Desktop AI Assistant com MCPs** | Assistente desktop para macOS que centraliza MCPs de Azure, Azure DevOps, SSH, terminal e firewall em uma única interface de IA. Atalho `⇧⌘Space` para abertura instantânea. | Electron · Anthropic SDK · Claude Agent SDK · MCP Protocol · SQLite |

| **AI Libre — Gerador de Conteúdo para Marketing** | Geração automatizada de textos, vídeos (Veo3 720p/1080p) e imagens para redes sociais a partir de palavras-chave, cores e referências visuais. | React 18 · OpenAI · Google Gemini · Google Veo3 |
| **Send — CLI de Tasks e Commits com IA** | CLI que analisa `git diff` com IA (Claude) e gera tasks humanizadas no Azure DevOps e commits técnicos no GitHub automaticamente. | Node.js · Claude Agent SDK · Azure DevOps API · simple-git |
